## Description
This 2-month UROP internship focused on investigating the yield-stress fluid behavior of a Silly Putty sample, as well as the respective roles of gravity and surface tension in relaxation and instability processes. The outputs available in this repository include:
(i) the internship report (PDF),
(ii) the defense slides (PDF), and
(iii) commented Jupyter notebooks for the pilot experiment and the two laser-sheet experiments.
